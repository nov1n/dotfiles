# GitHub Organization Repositories

Fuzzy search all repositories in a GitHub organization.

## Setup

1. Create a GitHub Personal Access Token with `repo` scope at https://github.com/settings/tokens
2. Install the extension in Raycast
3. Configure your GitHub token and organization name in extension preferences
4. Run "Refresh Repository List" to fetch all repositories
5. Use "Search Organization Repositories" to fuzzy search through them

## Commands

- **Search Organization Repositories**: Fuzzy search through cached repositories
- **Refresh Repository List**: Fetch latest repositories from GitHub and update cache

## Features

- Offline fuzzy search through all organization repositories
- View repository language and star count
- Open repository in browser
- Copy repository URL or clone command
